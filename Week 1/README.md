# Week 1: Environment Setup, Python Basics & Intro to OpenCV

Welcome to Week 1! This week is all about getting your development environment configured properly and understanding how to manipulate digital images and video streams.

---

## 💻 Part 1: Environment Setup

To minimize compatibility issues with various computer vision packages, we highly recommend stick to the versions below:

1. **Python 3.11:** Download and install from the [Official Python Website](https://www.python.org/). *Choosing 3.11 ensures compatibility with older MediaPipe APIs used in many popular tutorials.*
2. **VS Code:** Download and install from [Visual Studio Code](https://code.visualstudio.com/).
3. **Extensions:** Install the official **Python Extension** inside VS Code.

---

## 🐍 Part 2: Python Basics

If you are new to Python or need a quick refresher on foundational syntax, dive into these tracks:
* **Core Concepts:** Watch the first 3 lectures (Variables, Functions, Loops, and Conditionals) of [Harvard's CS50P Course](https://cs50.harvard.edu/python/).
