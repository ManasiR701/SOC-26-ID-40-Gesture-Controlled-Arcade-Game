> **Category:** Machine Learning · Computer Vision  
> **Mentor:** Sreshtha Lahiri · **Co-Mentor:** Manasi Rudsamudra

---

## What Is This?

This repository is the **one-stop resource hub** for a structured 8-week guided project where you build a fully working **gesture-controlled arcade game** (Snake/Flappy Bird) from scratch, controlled entirely by your hand movements through a webcam.

No prior experience with Computer Vision or MediaPipe is needed. Everything is taught from the ground up.

---

## What You'll Build

A real-time pipeline that goes:

**Webcam feed → Hand detection → Landmark extraction → Gesture classification → Game command → Game response**

By the end, you'll have a playable game that responds to your hand gestures live, with an optional HUD overlay showing the detected gesture and a quick calibration mode at startup.

---

## Tech Stack

| Tool | Purpose |
|---|---|
| `OpenCV` | Webcam feed, frame processing, drawing |
| `MediaPipe` | Hand landmark detection (21 points per hand) |
| `Pygame` | Arcade game engine |
| `Python` | Everything glued together |

---

## 8-Week Structure

Resources, notes, and code are uploaded **week by week** as the project progresses. Come back each week for the latest material.

| Week | Topic |
|---|---|
| Week 1 | Python & environment setup |
| Week 2 | OpenCV basics — webcam, frames, drawing |
| Week 3 | MediaPipe hand landmarks & visualization |
| Week 4 | Gesture logic & building the gesture module |
| Week 5 | Understanding the game code & input structure |
| Week 6 | Wiring gestures into the game |
| Week 7 | HUD, calibration & smoothing |
| Week 8 | Polish, demo video & final submission |

## Prerequisites

None. Previous exposure to Python or Computer Vision is appreciated but not required, just bring your enthusiasm to learn and explore!
